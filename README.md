# Medical Equipment Contracts

## Description
The interface offers a clear overview of ongoing contract, order status, and relevant notifications, empowering users to manage their procurement tasks effortlessly.

## Installation and Usage
* [Angular prerequisites](https://angular.io/guide/setup-local#prerequisites)
* Make sure you are running [External application.](https://github.com/nina-bu/isa-hospital-be)
* Navigate to the 'med-hospital' folder using the command line.
* Install dependencies with `npm install`.
* Run the application with `ng serve --open`.
